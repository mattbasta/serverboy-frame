A CSS framework that handles...

  * Layout
  * Typography
  * Forms
  * Commonly used controls
  * Code styling
  * Table styling
  * CSS reset
  * Print styling

The library itself is fully HTML5 compatible, with default styles and support for all of the latest HTML5 elements.
